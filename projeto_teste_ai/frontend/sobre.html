Sobre o Sistema

Este sistema foi desenvolvido como um painel administrativo para gerenciamento de usuários. Ele permite autenticação, visualização e controle de usuários cadastrados em uma aplicação web.

O objetivo do projeto é demonstrar a construção de um sistema completo utilizando backend e frontend integrados.

Tecnologias utilizadas

🐍 Python
🌶 Flask
🗄 SQLite
🌐 HTML
🎨 CSS
⚡ JavaScript
📊 Chart.js
